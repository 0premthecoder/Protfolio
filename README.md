# Sketch Minimal Portfolio (Vue 3)

A sketch-style, minimalistic, theme-switchable personal portfolio built using **Vue 3** and **Vite**.  
This site focuses on clean design, hand-drawn aesthetics, subtle animations, and interactive elements such as a roaming bug, custom cursor, and paint-trail effects.
